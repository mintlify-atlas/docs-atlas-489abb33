## What is TravelConnect?

TravelConnect is a full-stack travel social platform where users can discover trips, join travel groups, and communicate with fellow travelers through built-in trip chat. It connects travelers with trip creators and provides administrators with the tools to manage the community.

The platform is built with a **React + Vite** frontend and a **FastAPI** backend, using JWT-based authentication and a role-based permission system.

## Architecture overview

TravelConnect is structured as a monorepo with two top-level directories:

| Directory | Stack | Purpose |
|-----------|-------|---------|
| `Backend/` | Python, FastAPI, SQLAlchemy | REST API, database models, authentication |
| `FrontEnd/` | React 19, Vite, React Router v7 | Single-page application |

The backend exposes a REST API on port `8000`. The frontend dev server runs on port `5173` and proxies API calls to the backend.

## User roles

| Role | Catalan name | Capabilities |
|------|-------------|--------------|
| Traveler | Viatger | Browse trips, enroll/leave, chat, view profile, request promotion |
| Creator | Creador | All Traveler actions + create and edit their own trips |
| Admin | Admin | All Creator actions + manage all trips + manage users and promotion requests |

<Note>
  New accounts default to the **Traveler** role. Promotion to Creator requires submitting a request that an Admin approves.
</Note>
